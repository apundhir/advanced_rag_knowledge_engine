# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-05-24

### Added
- **Sentence Window Retrieval:** Precision context management strategies.
- **HyDE Integration:** Hypothetical Document Embeddings for query expansion.
- **Cross-Encoder Reranking:** Two-stage retrieval pipeline for high precision.
- **Documentation:**
    - Visual Architecture Diagrams in README and `docs/architecture.md`.
    - Deep Dive Concept Guides in `docs/concepts.md`.
- **Evaluation:** Integrated RAGAS metrics support and `evaluate.sh` script.
- **Infrastructure:** Docker and Docker Compose setup for easy deployment.

### Initial Release
- Complete operational source code for Advanced RAG pipeline.
- CLI tools for ingestion and querying.
- FASTAPI backend service.
